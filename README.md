# 💎 Kodas Bot V7

## Paleidimas lokaliai
```
pip install -r requirements.txt
python bot_v7.py
```

## Discloud hostingas
1. Pervardink `.env.example` į `.env`
2. Įdėk savo Discord token
3. Suspausk viską į ZIP
4. Įkelk į discloud.app
